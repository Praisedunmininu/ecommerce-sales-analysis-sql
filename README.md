# E-Commerce Sales Analysis (SQL Project)

## 📌 Project Overview
This project analyzes an e-commerce sales dataset using SQL to uncover insights on regional performance, product performance, customer behavior, and profitability.

The goal is to identify key business drivers behind sales and profit and provide actionable recommendations for business improvement.

---

## 📊 Objectives
- Identify top-performing regions based on sales and profit
- Analyze product performance across categories
- Understand customer purchasing behavior
- Evaluate average order value across regions
- Identify key drivers of revenue and profit

---

## 🛠️ Tools Used
- SQL Server
- GitHub

---

## 📌 Key Insights Summary
- West region is the top performer in sales and profit
- Electronics category generates the highest revenue
- Office category has the lowest performance due to limited product variety
- Customers in East show complementary buying behavior
- West has the highest average order value

---
